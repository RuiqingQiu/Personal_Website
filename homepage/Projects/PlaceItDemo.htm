PlaceIt (A Location Based Reminder)

This app is a class project that is done in a course of 10 weeks. The main idea is that, this app will track the user’s reminder and when user is close to the location of that reminder, it will send user a reminder to notify.

 

Besides the main functionality, this app implements a user login system. And it contains a different type of reminder, which is called categorical PlaceIt. The main idea is that users can specify what kind of places they want to visit. And as they move around, the app will find the closest place and remind them.

And when they are close to some position, they will receive a notification.  

 

This app uses Google App Engine to store all the data and users. And when different users log in, it will download the data that belongs to that user.
